# 나오시마 비전트립 가이드 — 기획서 (PRD)

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 나오시마 비전트립 가이드 |
| 목적 | 비전트립팀이 나오시마 섬을 사전에 이해할 수 있는 웹 가이드 |

## 2. 사용자 특성

- **연령대**: 50~60대 → 큰 글씨, 명확한 네비게이션, 단순한 UI
- **디바이스**: 모바일 우선 (링크 공유) + 데스크톱 프레젠테이션 겸용
- **기술 수준**: 일반 사용자 → 별도 설치 없이 브라우저에서 바로 열림

## 3. 콘텐츠 구조

### 섹션 1: 나오시마 프로젝트
- 나오시마 섬의 위치와 지리적 특성 (시코쿠 카가와현)
- 베네세 아트 사이트 나오시마 프로젝트의 배경
- 후쿠타케 소이치로와 안도 타다오의 비전
- 산업폐기물 섬 → 예술의 섬으로의 변화 타임라인

### 섹션 2: 주요 작가와 작품

#### 왜 이 작가들인가

- 후쿠타케 소이치로의 철학: "경제는 문화의 종(servant)"
- 안도 타다오와의 협업 이유: 자연과 건축의 공존, "빛과 침묵"의 건축
- 작가 선정 기준: 자연·명상·시간을 주제로 한 작가들을 섬이라는 맥락에 배치

#### 작가별 소개 (나오시마 작품 + 한국에서 만날 수 있는 작품)

- 쿠사마 야요이 — 나오시마: 노란 호박, 빨간 호박 / 한국: 대구미술관, 서울 루이비통 메종 등 전시 이력
- 안도 타다오 — 나오시마: 지중미술관, 이우환미술관, 베네세하우스 / 한국: 뮤지엄산(원주), 본태박물관(제주), 글래스하우스(제주)
- 클로드 모네 — 나오시마: 수련 (지중미술관 소장) / 한국: 전시 이력 비교
- 월터 드 마리아 — Time/Timeless/No Time
- 제임스 터렐 — 나오시마: Open Sky / 한국: 뮤지엄산 제임스 터렐관(원주)
- 이우환 — 나오시마: 이우환미술관 / 한국: 부산시립미술관 이우환 공간, 국립현대미술관 전시 이력
- 아트하우스 프로젝트 (혼무라 마을)

### 섹션 3: 선교적 관점 — 시코쿠에서 보는 일본

#### 시코쿠의 역사적 사건과 선교적 의미

- 시코쿠 88개소 순례길 (헨로): 1200년 전통의 영적 순례 문화 → 일본인의 영적 갈망과 "구도자" 정체성
- 세토내해의 역사: 해상 무역과 해적, 섬 공동체의 고립과 자립 → 복음이 닿기 어려웠던 지리적 이유
- 메이지 유신과 시코쿠: 근대화에서 소외된 지역 → 오늘날까지 이어지는 인구 감소와 고령화
- 나오시마의 산업폐기물 역사 (1910~1980년대 제련소) → 상처와 회복의 서사
- 시코쿠 기독교 역사: 초기 선교사들의 활동, 현재 교회 현황

#### 선교적 관점으로 매칭

- 순례 문화 → "길이요 진리요 생명" — 일본인의 영적 탐색을 복음으로 연결하는 접점
- 섬의 회복 (폐허 → 예술) → 하나님의 회복 사역의 메타포
- 고령화 사회 → 교회의 돌봄 사역 가능성
- 비전트립에서 기도할 포인트

### 섹션 4: 실용 정보

#### 지도와 교통

- 섬 지도 (주요 미술관, 항구, 식당 위치)
- 교통수단 (페리: 우노항/다카마츠항, 섬 내 버스/자전거)

#### 4월 중하순(4/18 전후) 방문 가이드

- 기후: 벚꽃 막바지~신록 시작, 평균 15~20°C, 세토내해 온화한 해양성 기후
- 자연: 섬 전체가 초록으로 전환되는 시기 — 안도 타다오 건축의 콘크리트와 신록의 대비가 가장 극적
- 관광 시즌: 골든위크(4/29~) 직전 → 비교적 한산한 마지막 시기, 여유롭게 관람 가능
- 빛의 변화: 봄의 부드러운 자연광 — 지중미술관 모네 작품실, 터렐 작품의 빛 체험에 최적
- 영적 의미: 부활절(2026년 4/5) 직후의 시기 — "죽음 후의 새 생명"이라는 부활의 메시지와 폐허에서 회복된 나오시마의 서사가 겹침
- 실용 팁: 월요일 휴관 주의, 우천 시 실내 미술관 중심 동선 추천

## 5. 기술 요구사항

- 정적 사이트 (서버 불필요)
- 모바일 반응형 (mobile-first)
- 빌드 도구 최소화 (가능하면 순수 HTML/CSS/JS 또는 가벼운 프레임워크)

- 이미지 최적화 (모바일 데이터 고려)

- 한국어 전용
